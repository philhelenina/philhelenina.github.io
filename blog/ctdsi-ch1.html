[Book Summary - CtDSI] Cracking the Data Science Interview Ch. 1

January 19, 2023

What is Data Science?

Data science is defined as "deriving insights from data. These insights are then used to provide value to a business."

Main factors driving data science:

  • Explosion of data
  • Technological advances
  • Success stories

Data Science ≠ Machine Learning

Machine learning refers to algorithms identifying patterns with minimal human intervention.

Supporting disciplines:

  • Operations Research
  • Statistics

What Makes a Good Data Scientist?

  • Identify relevant questions
  • Acquire and clean data
  • Analyze data
  • Communicate findings
  • Convert results into solutions

Data Science Process Workflow

Steps: Specify Objective → Data Acquisition → Explore Data → Establish Baseline → Model Data → Analyze Results → Communicate Findings → Iterate

Data Science Deliverables

  • Prediction
  • Forecasts
  • Anomaly Detection
  • Recognition
  • Optimization
  • Segmentation
  • Recommendations

Writing a Great Data Science Resume

  • One-page format
  • Relevant coursework and skills
  • Unique projects
  • Quantifiable accomplishments
  • Customization for specific jobs

Data Science Interview Topics

  • Probability & Statistics
  • Computer Science
  • Machine Learning
  • Data Engineering
  • Domain Knowledge
  • Behavioral Questions

Data Science Interview Process

  • Coding Challenge
  • HR Screen
  • Technical Screen
  • Take Home Project
  • Onsite Interview
  • Offer & Negotiation

Behavioral & Fit Questions

Use STAR method (Situation, Task, Action, Result)
